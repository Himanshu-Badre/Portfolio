Internship Project: Cab Service Management


As a passionate and dedicated React.js developer, I embarked on an enriching journey during my internship at StarCenturi Pvt. Ltd. My mission was to elevate the Cab Service Management system to new heights, optimizing the user experience and refining cab allocation processes.

Project Highlights User-Centric Development:

Developed and maintained the user component with a razor-sharp focus on authentication and real-time activity tracking. Ensured that users enjoyed a seamless and secure experience.

Dynamic Dashboard:

Crafted an engaging real-time dashboard that provided a bird's-eye view of active users, their status, and streamlined cab assignments, right down to company specifics.

Cutting-Edge Tech Stack: Leveraged a cutting-edge technology stack, including React.js, HTML5, CSS3, and JavaScript, to create a dynamic and responsive user interface.

Form Mastery:

Employed Formik and Yup to masterfully manage forms and validation, ensuring data accuracy and reliability.

Seamless Data Interaction:

Integrated Axios for frictionless API communication, facilitating swift data exchange between the front end and back end.

UI Elegance:

Enhanced the aesthetics and functionality of the project using Material-UI and React-Bootstrap, delivering an intuitive and visually stunning application.

Version Control Pro:

Collaborated seamlessly with the team, harnessing the power of Git for efficient version control and streamlined code management.
